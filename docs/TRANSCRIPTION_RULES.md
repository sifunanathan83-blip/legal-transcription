# TranscribeMe Legal Formatting Rules

## Overview

This document outlines the legal transcription formatting standards used by the Legal Transcription App, following TranscribeMe conventions for court-ready transcripts.

## Speaker Labels

### Format

All speaker labels must be in ALL CAPS, followed by a colon:

```
THE COURT:
MR. ATTORNEY:
MS. ATTORNEY:
MR. WITNESS:
THE REPORTER:
```

## Q&A Format (Colloquies)

Questions and answers use "Q:" and "A:" format:

```
Q: Tell us what you observed.
A: I saw a blue car turn the corner.
Q: What time was this?
A: Approximately 3 p.m.
```

## Timestamps

### Time Notation Format

Timestamps appear in square brackets and include time notation:

```
[HH:MM:SS]
(Time notation: H:MM:SS a.m./p.m.)
```

### Example

```
MR. ATTORNEY: Your Honor, I move to suppress the evidence.
[10:45:32]
(Time notation: 10:45:32 a.m.)
THE COURT: Overruled.
```

## Objections and Rulings

### Common Objections

- Objection, assumes facts not in evidence.
- Objection, compound question.
- Objection, vague and ambiguous.
- Objection, calls for speculation.
- Objection, hearsay.
- Objection, improper foundation.

### Rulings

```
THE COURT: Sustained.
THE COURT: Overruled.
THE COURT: Withdrawn.
THE COURT: Noted.
```

### Example

```
Q: What did the defendant say to you?
MR. ATTORNEY: Objection, hearsay.
THE COURT: Sustained.
```

## Procedural Markers

Procedural events are noted in parentheses:

```
(Whereupon, a brief recess was taken.)
(Whereupon, the proceeding was adjourned.)
(Whereupon, the witness was sworn.)
(Whereupon, the jury was excused.)
```

## Court Reporter Notations

Special notations appear in parentheses:

```
(Pause)
(Brief pause)
(Long pause)
(Inaudible)
(Unintelligible)
(Off the record)
(On the record)
(Crosstalk)
```

## Exhibits

### Reference Format

```
Q: I'm showing you what's been marked as Exhibit 1. Do you recognize it?
A: Yes, I do.
Q: What is Exhibit 1?
A: That's the purchase agreement.
```

## Phonetic Spellings

### Unknown or Unusual Terms

```
[phonetic: BLURGENSTEIN]
[phonetic: MONEE-PULL or MON-O-POLE]
```

- [ ] Q&A format used consistently
- [ ] Timestamps placed at 60-second intervals
- [ ] All objections have rulings
- [ ] Procedural markers included
- [ ] Exhibits properly referenced
- [ ] Phonetic spellings marked
- [ ] Page numbers included
- [ ] Professional appearance maintained
